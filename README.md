# Laundry Services Mobile Hamburger Menu

## Description

This project implements a responsive hamburger menu for the mobile version of the Laundry Services web application. In the previous assignment, the navigation links were hidden on smaller screens. This task adds a hamburger icon that allows users to access the navigation menu on mobile devices.

## Features

* Responsive navigation bar
* Hamburger menu icon visible only on mobile screens
* Navigation links hidden on mobile view
* Slide-out menu from the right side of the screen
* Menu visibility controlled using CSS `:focus` pseudo-class
* No JavaScript used (implemented entirely with HTML and CSS)

## Files Included

* **index.html** – Structure of the webpage including navbar, hero section, and hamburger menu.
* **index.css** – Styling for the layout, responsive design, and hamburger menu behavior.

## How to Run

1. Download or clone the project folder.
2. Open the folder in a browser or code editor.
3. Open `index.html` in any web browser.
4. Resize the browser window or use developer tools to test mobile view.
5. Click the hamburger icon to display the menu.

## Technologies Used

* HTML
* CSS (Flexbox, Media Queries, Pseudo-classes)
